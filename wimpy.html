If you have more than one user account on your computer,fast user switching is an easy way for another person to log on to the computer without logging you off orclosing your programs

Shortcuts to use in the desktop include; windows key and A- use action centre , windows key and D- refresh the computer, windows key and tab- to switch the apps and window key+E- toopen the file explore.
